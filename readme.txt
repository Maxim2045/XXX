Пароль на авторизации: 1

Это выпускная квалификационная работа студента 4-го курса.
Проект выполнен в среде разработки WPF .NET Framework на языке программирования C#.

Из деканата поступил запрос на программу для автоматического внесения информации о студентах 
в экзаменационные ведомости.

Данные о студентах храняться в Excel таблицах, необходимо перенести в Word шаблон.

	Что есть в проекте:
	1. Для работы с Excel(копирование) и Word(внесение данных) была ис-пользована библиотека OpenXML.
	2. Имеется реализация окна редактирования Word файлов с помощью контрола RichTextBox. Для переноса информации из документа на контрол использована библиотека Microsoft.Office.Interop.Word(реализовано в виде copy-paste).
	3. Реализованы функции сохранения (FileSteam) и печати (System.Printing) текста с RichTextBox.
	4. В текстовом редакторе имеется выбор шрифта и его размера; подчер-кивание, курсив и жирный шрифт.
	5. Имеется БД для хранения данных о студентах. Написана в среде SQLite. Создавалась программно с помощью библиотеки System.Data.SQLite. Взаимодействие с посредством EntityFramework. Данные о студентах вносятся вручную, можно использовать в качестве списка злостных должников.
	6. Для формирования приятного интерфейса были использованы GUI библиотеки FluentWPF и MaterialDesign.
	7. Код прокомментирован, исключения обработаны
	Имеются попытки в ООП, логика из кнопок вынесена в классы, все свя-занное с БД то же имеет классовую структуру. 
	Если возникают вопросы почему не WPF NET.Core или даже ASP MVC .NET Core, то тут два соответственно объяснения:
	1) у заказчика все машина на Windows, кроссплатформенность не тре-буется.
	2) а на счет web приложения, не захотел разбираться с реализацией диа-логовых окон и передаче информации из Word файла на какой-нибудь онлайн редактор. Нет достаточной информации на эту тему, а для WPF подобные ре-ализации встречались в рамках обучающего материала. А так да, API было бы хорошо.

Для тех, кто будет работать с схожей темой проект будет полезен.

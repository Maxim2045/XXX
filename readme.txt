Пароль на авторизации: 1

Это выпускная квалификационная работа студента 4-го курса.
Проект выполнен в среде разработки .NET Framework(WPF) на языке программирования C#.

Из деканата поступил запрос на реализацию приложения для автоматического внесения информации о студентах 
в экзаменационные ведомости.

Данные о студентах в деканате храняться в Excel файлах, необходимо из них перенесить данные в Word шаблон ведомости.

	Что есть в проекте:
	1. Для работы с Excel(копирование) и Word(внесение данных) была использована библиотека OpenXML.
	2. Имеется реализация редактирования Word файлов с помощью элемента управления RichTextBox. Для переноса информации из документа на элмент управления использована библиотека Microsoft.Office.Interop.Word(реализовано в виде copy-paste).
	3. Реализованы функции сохранения и печати текста с RichTextBox.
	4. В текстовом редакторе имеется выбор шрифта и его размера; подчеркивание, курсив и bold шрифт.
	5. Имеется база данных(БД) для хранения данных о студентах. Написана в среде SQLite. Создавалась программно с помощью библиотеки System.Data.SQLite. Взаимодействие с базой(добавление, редактирование, удаление) данных из БД осуществлется с помощью EntityFramework. Данные о студентах в БД вносятся вручную пользователем.
	6. Для формирования приятного интерфейса были использованы GUI библиотеки FluentWPF и MaterialDesign.
	7. Код прокомментирован, исключения обработаны.
	8. Реализованна классовая стуктура в рамкам ООП. 
	Если возникают вопросы почему не WPF .NET Core или даже ASP MVC .NET Core, то:
	1) у заказчика все машина на Windows, кроссплатформенность не требуется.
	2) web приложение, на мой взгляд, менее подходит для работы с локальными файлами.

Для тех, кто будет работать с схожей темой проект будет полезен.
